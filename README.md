# 🥊 KickboxingTrainer: HIIT Performance
### **Custom Android Implementation for High-Intensity Interval Training**

## 📌 Project Context
Developed to solve the "Screen-Timeout" friction point during high-intensity sessions. This app utilizes **Kotlin** and **Android Jetpack** to maintain active-state governance while executing complex interval timers.

## 🚀 Key Features
* **Wake-Lock Management:** Engineered custom logic to override system sleep-cycles during active training blocks.
* **State Governance:** Utilizes background threading to ensure timer accuracy remains consistent even during high CPU load.
* **Clean Architecture:** Developed with a focus on separation of concerns between the HIIT logic and the UI layer.
